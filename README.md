🏰 Dungeon RPG (Java)

A simple console-based RPG game built in Java where the player battles randomly generated enemies, manages health, and decides whether to fight or escape.

---

🧩 Game Overview

- You enter a dungeon and encounter random enemies 
- Each turn you can:
  - Attack
  - Drink a health potion
  - Run away
  - Defeating enemies gives a chance to receive health potions 
  - The game continues until you die or choose to exit
    
---

🎮 Features

- ⚔️ Turn-based combat system 
- 👾 Random enemy encounters (Skeletons, Zombies, Warriors, Monsters) 
- ❤️ Health potion system with drop chance 
- 🏃 Ability to run from battles 
- 🔁 Continuous game loop until player exits or dies 
- ✅ Input validation for user choices 

---

🛠 Technologies Used

- Java 
- `Scanner` (for user input) 
- `Random` (for combat mechanics and enemy spawning)
